MzanziMarket Setup Steps

1. Install XAMPP.
2. Open XAMPP Control Panel and start Apache and MySQL.
3. Copy the MzanziMarket_Project folder into C:\xampp\htdocs\
4. Open http://localhost/phpmyadmin
5. Click Import and choose sql/database.sql
6. Open http://localhost/MzanziMarket_Project/index.php
7. Demo logins:
   Admin: admin@mzanzimarket.co.za | password123
   Seller: seller@mzanzimarket.co.za | password123
   Buyer: buyer@mzanzimarket.co.za | password123

Important: This is a prototype. For hosting, update includes/db.php with the hosting database name, username and password.
