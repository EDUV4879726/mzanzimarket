MzanziMarket C2C E-Commerce Platform

Project Overview
MzanziMarket is a PHP and MySQL based C2C e-commerce platform developed for informal South African traders and buyers. The platform allows buyers to browse products, sellers to manage product listings and administrators to monitor the platform.

Technologies Used
PHP
MySQL
HTML
CSS
JavaScript
Bootstrap
XAMPP

Features
User Registration and Login
Buyer Product Browsing
Product Management
Seller Dashboard
Administrator Dashboard
Product Approval System

Database
The MySQL database export file is included as:
database.sql

Running the Project
1. Install XAMPP.
2. Start Apache and MySQL.
3. Place the project folder inside the htdocs directory.
4. Import database.sql into phpMyAdmin.
5. Open the project using localhost in a web browser.
